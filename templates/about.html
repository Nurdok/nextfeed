About nextfeed

nextfeed is about minimalism.
I created nextfeed for me. With the death of Google Reader (RIP), a plethora of alternative feed readers was created,
but I craved that one simple feature - just reading my feeds.


About Me

Technological Mumbo-Jumbo

I created nextfeed mainly as a learning experience.
Because of that, I feel obliged to share what I learned and how nextfeed is implemented.

Frontend

  • Open-ID Integration is done with django-openid-auth.
  • Design / Functionality is done with bootstrap and some Angular.js.
  • Copy to Clipboard is done with ZeroClipboard.

Backend

  • Web Framework is done with Django.
  • Asynchronuous Feed Updates is done with Celery.
